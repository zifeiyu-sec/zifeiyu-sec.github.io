时间盲注

概述

时间盲注，也叫条件时间盲注，是一种 SQL 注入攻击方式。

攻击原理

时间盲注原理是，攻击者通过控制输入数据，将多个 SQL 语句联合成一个 SQL 语句，从而执行多个 SQL 语句，获取数据库中敏感信息。
